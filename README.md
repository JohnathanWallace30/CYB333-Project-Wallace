# CYB 333 – Cybersecurity Dataset Analysis with Jupyter Notebook

## Project Overview

This project analyzes a real cybersecurity dataset using Python and Jupyter Notebook. The purpose of the project is to understand how security-related data can be explored and visualized, and how automation tools can support cybersecurity analysis. Instead of only working with theory, this project focuses on hands-on data analysis using a real intrusion detection dataset.

The dataset used in this project is the NSL-KDD dataset, which includes labeled network traffic data. Each record represents a network connection and is labeled as either normal traffic or a specific type of attack.

---

## Project Objectives

- Analyze a real-world cybersecurity dataset
- Practice data analysis using Python
- Explore attack patterns and normal network behavior
- Use visualizations to better understand security data
- Demonstrate how Jupyter Notebook supports security automation tasks

---

## Features

- Automatic download of the NSL-KDD dataset within the notebook
- Data exploration and preprocessing using Pandas
- Grouping of raw attack labels into higher-level attack categories
- Statistical analysis of network traffic features
- Visualizations such as bar charts, distributions, and correlation heatmaps
- Clear markdown explanations throughout the notebook

---
